About

Ngẫu - Just some random words.

Some introduction

Welcome to my blog, a minimal website I created for thought sharing and personal reflection. If you have reached here, you should have known me a little bit. But still, I'll make a quick introduction for those who don't. My name is Nghĩa, a computer science student of class 2024, VinUniversity - a newly established university in Vietnam. I suppose myself to be an extrovert as I find myself energetic and enthusiastic working with people, especially in clubs or teams. Still, you may find me being quiet in indistinct chatting, but just because I'm trying to keep up with what is going on :v. Photography and magic are apparently what you will see me doing a lot as I enjoy capturing the wonders of the scenery as well as people's expressions (pun intended). And well, I do have a dedicated menu for photography on this blog, so... Check it out!

Some reasons

Sharing used to be hell for me, and now it still is. I find myself awkwardly trying to express opinions I don't know much about - a little bit of insecurity. So yeah, I started this blog to start sharing anything I want as a step for developing myself. A post written is also a chance for improving my writing, or you can say my old nemesis. College courses, especially the HASSes, have actually inspired me to write and discover more. For the first time, I can put my thoughts into fluent flows of words, which I hope to keep them going. And there you are, a simple blog.

Some backgrounds

I hosted this blog on Github Pages for one simple reason: it's free. Also, the domain looks professional :D. However, I also want to make the writing and managing process more interactive on my side, so I need some kind of CMS platform. Boom, Ghost + Github Pages. I make a local installation of Ghost CMS, use GSSG (Ghost to Static Site Generator) to generate a static version of my blog, and push all the content to Github Pages using a simple automated script.

That's pretty much about it. If needed, you can reach out to me via:
